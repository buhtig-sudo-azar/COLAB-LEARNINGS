# COLAB-LEARNINGS

Учебный курс по машинному обучению и глубокому обучению в формате интерактивных Google Colab ноутбуков.

## Философия курса

Каждый ноутбук — это не просто теория, а полноценная рабочая среда:

- **Весь код запускается прямо в Colab** — ничего не нужно устанавливать локально
- **FastAPI-серверы** в каждом ноутбуке позволяют тестировать модели через HTTP API
- **Интерактивные виджеты** (слайдеры, кнопки) дают возможность менять параметры на лету
- **Каждая строка кода прокомментирована** — вы автор, а не потребитель

## Структура папок

```
COLAB-LEARNINGS/
├── ИИ/                            # основной ML/DL курс (ноутбуки 01-15)
│   ├── 01_what_is_ml.ipynb
│   ├── 02_pytorch_intro.ipynb
│   ├── ...
│   ├── 15_mlops.ipynb
│   └── README.md
└── PEN/                           # security / fraud detection
    └── 16_splunk_fraud.ipynb      # Splunk Fraud Detection
```

- **`ИИ/`** — основные учебные ноутбуки курса по ML и DL (модули 01-15)
- **`PEN/`** — отдельная папка на уровне `ИИ/`, содержащая ноутбуки по security, fraud detection и аномалиям. Ноутбук 16 (Splunk Fraud Detection) вынесен сюда, так как тематически относится к security, а не к базовому ML/DL курсу.

## Как пользоваться

1. Откройте ноутбук по ссылке в Google Colab
2. Запустите первую ячейку для установки зависимостей
3. Следуйте пояснениям и запускайте ячейки последовательно
4. Меняйте параметры через виджеты и наблюдайте результаты
5. Запустите FastAPI-сервер и тестируйте через HTTP-запросы

> **Ноутбук 16** расположен в отдельной папке [`PEN/`](../PEN/) (на уровне `ИИ/`) — там собраны материалы по security и fraud detection.
