# COLAB-LEARNINGS 🧠

Обучающие блокноты по машинному обучению и нейронным сетям.

## Структура

| Файл | Тема |
|------|------|
| [`01_pytorch_intro.ipynb`](ИИ/01_pytorch_intro.ipynb) | Введение в PyTorch: тензоры, autograd, nn.Module |

## Как использовать

1. Нажмите на ссылку ноутбука
2. Откройте в Google Colab (кнопка "Open in Colab")
3. Выполняйте ячейки по порядку
